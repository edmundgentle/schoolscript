Variables

Variables are bits of data which are stored during the running of the programming. They can be used to pass information between different instructions.

  • End Tutorial
  • Next

Variables have 3 key parts. The first part is the name. All variables must have a name. In PHP, the name must start with a dollar sign ($) and must then have at least one alphabetic (upper case or lower case), numerical or underscore (_) character. This is how you will access your variable's data.

  • End Tutorial

  • Next

In PHP, the standard is to use lowercase characters with underscores in between each word. For example:

$my_variable
$bobs_variable
$total
$score

The second part of a variable is its type. In PHP this isn't very important (PHP is a "weakly typed" language meaning that the type of variable doesn't matter).

A variable can have a number of different types. This is based on the type of information it stores. This can be an integer (a whole number), a double (any type of number - including decimals), string (text) and Boolean (either true or false). There are more types which we will discuss later on.

The final part of a variable is its value. This can change at any point within the script.

The variable's name can be used to read and write the value. For example, you can set the value 5 to a variable called $number:

<?php
$number = 5;
?>

Or you can output the value of a string variable using echo:

<?php

First of all, we will take another look at Boolean variable types. These can be either true or false. We can set them using the equals sign, as we did with other variable types:

<?php
$my_bool = true;
$other_bool = false;
?>

We can also set Boolean values using expressions. These compare values. We have four main Boolean expressions which can be used.

  • End Tutorial
  • Previous
  • Next

The first is less than. To use this, we can just use the less than sign:

<?php
$my_bool = ( 5<3 ); //false
?>

We can also do "less than or equal to" just by following the less than sign with an equals sign:

<?php
$my_bool = ( 5<=6 ); //true
?>
  • End Tutorial
  • Previous
  • Next

The second is greater than. This behaves in the same way as less than:

<?php
$my_bool = ( 6>5 ); //true
?>

We can also do "greater than or equal to" just by following the greater than sign with an equals sign:

<?php
$my_bool = ( 2>=6 ); //false
?>
  • End Tutorial
  • Previous
  • Next

The third is equal to. Since we use the equals sign to set a variable's value, we need something different for comparing variables. For this, we use two equals signs:

<?php
$my_bool = ( 5 == 5 ); //true
?>

If both of the values are the same, the value is true, otherwise it's false.

  • End Tutorial
  • Previous

The final expression is "not equal to". If the values are different, then it is true, otherwise it's false. To do "not equal to", we use an exclamation mark followed by an equals sign (!=).

<?php
$my_bool = ( 4!=6 ); //true
?>

We can also combine expressions using AND and OR.

  • End Tutorial
  • Previous
  • Next

AND looks at a set of values. If any one of them is false, then the value is false, otherwise it is true.

<?php
$value = ( true AND true ); //true
$value = ( true AND false AND true ); //false
?>
  • End Tutorial
  • Previous
  • Next

OR also looks at a set of values. If any of the values within the set are true, then the output is true, otherwise the output is false.

<?php
$value = ( true OR false ); //true
$value = ( false OR false OR false ); //false
?>

Expressions can be "nested" (put inside one another) using brackets. They work just like mathematical equations.

If a variable is an integer (whole number) or double (any number) then you can do mathematical tasks with it. For example, if a 15% tip is charged at a restaurant, then we could calculate the price in the following way:

<?php
$price = 24.98;
$price = $price * 1.15;
?>
  • End Tutorial
  • Previous
  • Next

We multiply the price by 1.15 to add 15% to the price. The asterisk symbol (*) is used for multiplication and the forward slash symbol (/) is used for division. We can use plus (+) and minus (-) as normal.

  • End Tutorial

  • Next

We can also use a couple of "quick expressions" for incrementing (adding one) and decrementing (subtracting one) from variables. These are ++ for adding one and -- for subtracting one. These are used in the following way:

<?php
$my_num = 5;
$my_num++; //my num is now 6
$my_num--; //my num is now 5 again
?>
  • End Tutorial

If you are using strings (text), you can connect two strings to each other (called concatenation) using the full stop (.) character:

<?php
$name = "Edmund";
echo "Hello " . $name;
?>
