Before we look at how to create a loop, we will warn you about infinite loops. You will probably create an infinite loop at some point - everyone does! These are often caused by programmer error. Basically, all loops have a "loop condition". If the loop condition never finishes, then the loop will continue forever. We will warn you of how these can be caused and prevented for each type of loop.

We are going to look at three ways of iterating (looping through code) which all behave a little differently.

The first is called a FOR loop. A FOR loop starts at a number and counts to another number. It can count up and down in any sequence.

For a FOR loop, you need three main pieces of information. The first is the starting value. Where do you want to start counting from?

The second is the end value. Where do you want it to stop counting?

The third is the change at each step. Do you want it to go up one each time, or go down one? Or go up three?

In this example, we are going to count from 1 to 10, going up by one each time.

<?php
for($x = 1; $x <= 10; $x++ ) {
   echo $x . "\n";
}
?>

If your loop condition is never met, then it will never actually run the code. For example, if you start your loop from 1 and say "while it's less than 1", then the loop will never run, because 1 is never less than 1.

The next type of loop is a WHILE loop. A sample while loop is:

<?php
$num = 0;
while( $num < 5 ) {
   $num++;
}
?>
  • End Tutorial
  • Previous
  • Next

A while loop looks at a condition, like an IF statement. If the condition is true, then it runs the code within the loop. Once it's run this code, it will check the condition again. If it's true, it will continue to run the code until the condition isn't true.

  • End Tutorial
  • Previous
  • Next

The code within the loop should change whatever is being checked in the condition. In the example, $num is being checked in the condition, so we must change $num in the code. If we don't then we would get an infinite loop.

  • End Tutorial
  • Previous
  • Next

Like the FOR loop, if the condition isn't true before the loop starts, then the code inside the loop will never run. A WHILE loop may run zero or more times.

  • End Tutorial

A WHILE loop can be used for solving a factorial problem. Factorials are where you take a number (let's say 5) and you multiply it by all the previous numbers. So the factorial of 5 is 1x2x3x4x5. We can write this as a program like so:

<?php
$num = 5;
$total = 1;
while( $num>1 ) {
   $total = $total * $num;
   $num--;
}
?>

The final type of loop is a DO WHILE loop. This is very similar to a WHILE loop, although a DO WHILE will run the code at least once.

<?php
$x = 0;
do {
   $x++;
} while ($x < 5 );
?>

This loop is laid out a little differently to the other two. It starts with the keyword "do" followed by the code in braces. Then there is the keyword "while" followed by the loop condition. This loop condition works in the same way as a WHILE loop although it checks the condition after running the code rather than before it.
