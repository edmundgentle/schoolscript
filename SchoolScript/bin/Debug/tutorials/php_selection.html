Selection

When you write a program, you might not want some bits to run every time. For example, you may want to charge free delivery on orders over £25. To do this, we use IF statements.

  • End Tutorial
  • Next

To solve this problem we would use the following code:

<?php
if( $total > 25.00 ) {
   $delivery = 0.00;
}else{
   $delivery = 5.99;
}
?>

This code says that if the total cost (in the variable $total) is greater than (>) 25.00, then delivery costs 0.00, otherwise delivery costs 5.99.

The IF statement may look a little confusing, but it's made up of a few very simple parts. First of all, every IF statement begins with the keyword "if". This is followed by brackets which contain a condition.

  • End Tutorial
  • Previous
  • Next

A condition is a Boolean expression. See the Variables section for more on Boolean expressions. To recap, this could be less than, greater than, equal to or not equal to as well as combinations using AND and OR. For the IF to run the code within it, the condition must be "true".

  • End Tutorial
  • Previous
  • Next

The next thing to point out are the curly brackets ( { } ) which we call braces. They mark out blocks of code. Code within braces is indented (by pressing the TAB key on your keyboard once) to make it easier to read. Anything within the braces following the IF statement and condition will run if the condition is "true".

The final part included in the example above is the "else" condition. This means that if the condition is false, it will run the code following the else. An else condition doesn't need to be included in an IF.

There is a lot to take in here, so we'll take a quick look back at IF statements before we move on. We'll use some examples:

<?php
if( 5 == 2 ) {
   echo "Five is the same as two!";
}
?>

This compares the values 5 and 2. If they are the same (which they aren't) then the code will output a string.

<?php
$total = 50.47;
$vat = 0.00;
$delivery = 4.99;
$vat_exempt = false;
$free_delivery = false;

if( $total > 25 ) {
   $delivery = 0;
   $free_delivery = true;
}

if($vat_exempt == false) {
   $vat = ($total + $delivery) * 0.20;
}
?>

This example checks the price of the order. If the order is over £25, then the customer receives free delivery. If the customer is not VAT exempt, then they are charged 20% of their order as VAT.

We can take IF statements one level further using "elseif". At the moment, we can try using one condition. If it doesn't match, then it just uses the code within the "else" block if it has been provided.

  • End Tutorial
  • Previous
  • Next

Using "elseif", we can have the program check for different conditions if the first one doesn't match. For example:

<?php
$fav_colour = "blue";

if( $fav_colour == "red" ) {
   echo "My friend likes red too!";
}elseif( $fav_colour == "blue" ) {
   echo "Same favourite colour as me!";
}elseif( $fav_colour == "green" ) {
   echo "Grass is green...";
}else{
   echo "I don't know anything about that colour.";
}
?>
  • End Tutorial
  • Previous
  • Next

Here we have different options for each of three colours. If the favourite colour is red, blue or green then it displays a relevant message, otherwise it shows a message saying that it doesn't know anything about a colour.

An IF statement can have as many "elseif"s as it needs, but it can only have one "if" and one "else".
