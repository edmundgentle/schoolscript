Dates and times within PHP are stored as UNIX timestamps. This is a numeric representation of a date. It is simply the number of seconds since 1st January 1970. So, 1st January at 1:00am would be 60(mins)x60(seconds) = 3,600.

We can output data in a useful way (often for passing between websites) using JSON. This stands for JavaScript Object Notation and is used widely by most companies for sharing data.

  • End Tutorial
  • Previous
  • Next

We can, for example, turn an array into a JSON string and output it in the following way:

<?php
//create a random array
$my_array = array(
   'foo' => 'bar',
   'hello' => array(
      'name' => array(
         'Jim',
         'Bill'
      ),
      'world'
   ),
   'Something else'
);
echo json_encode( $my_array );
?>

We can also read in JSON data using "json_decode" and "file_get_contents" to read in a file.

  • End Tutorial
  • Previous
  • Next

This is useful for pulling in live data from another source. An example of this is for a tool which reads in the weather for a specific location:

<?php
$location_code = 'UKXX0085';
$url = 'http://query.yahooapis.com/v1/public/yql?q=select%20item%20from%20weather.forecast%20where%20location%3D%22' . $location_code . '%22&format=json';
$file_contents = file_get_contents( $url );
$weather_data = json_decode( $file_contents, true );
if( is_array( $weather_data ) ) {
   if( isset( $weather_data['query']['results']['channel']['item']['condition']['text'] ) ) {
      $condition_text = $weather_data['query']['results']['channel']['item']['condition']['text'];
      echo 'The weather in London is ' . $condition_text;
   }
}
?>

Please note, this code worked at the time of writing. Because it uses Yahoo!'s data, they may change the permissions of the data or even stop supplying it. Because of this, we can't guarantee this will work, but Yahoo! are a trusted and reliable source so will likely keep the data available for some time - thank you!
