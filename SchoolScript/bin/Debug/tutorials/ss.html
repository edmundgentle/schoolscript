SchoolScript

SchoolScript has been designed to be as simple to use as possible, but using any piece of software can be difficult, so we've got this guide to talk you through it!

First of all, there are two panels you should be aware of. To the left, there is the code editor. This is where you write your code which you then run. Below, there is the output panel. This is where the program is executed when you press "Run".

You can run a number of different scripting languages in SchoolScript. You can select the language of your file by going to "File Type" at the top of the window then selecting the appropriate language from the drop down menu.

To run the code you write, go to "Run" in the top menu then click "Run". The "Stop" option in the "Run" menu is placed there in case your code has a mistake and doesn't end when it should (for example, in an infinite loop).

Within the tutorials, we provide you with definitions, such as "keyword". These are highlighted in green with an underline so they are easy to see. You can click on them for a full definition.
