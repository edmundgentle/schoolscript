Functions

All programming languages use something called functions. In other languages they may be referred to as methods, subroutines or procedures. Whilst they may technically differ slightly, the core idea is the same.

  • End Tutorial
  • Next

A function is a collection of instructions which have been given a name. They do a very specific task, for example, finding the largest number from a set of numbers. They can be used as many times as you like within your code and can make doing a very common, repetitive task much quicker and easier.

Like most things we've discussed, a function has three main parts! The first part is a name. Like a variable, this can be any alphanumeric character or an underscore (_) character. The standard in PHP is to use lowercase and to separate words with underscores.

The second part of a function are its parameters. These are variables which is passed into a function. You can have as many variables as you like.

The third part is the code within the function. This can pass data back but it doesn't have to. As soon as your function passes data back, it stops running, regardless of whether it's reached the end or not.

Okay, so let's take a look at an example. Let's write a function which gives you the highest number out of two numbers.

<?php
function max_of_two( $num1 , $num2 ) {
   if ( $num1 > $num2 ) {
      return $num1;
   }else{
      return $num2;
   }
}
?>

So if we have a look at how a function is constructed, we can see that it starts with the keyword "function" followed by the name of the function.

So we've written a function. How do we use it? It's pretty simple!

<?php
$max = max_of_two( 54, 83 );
?>

We simply write its name followed by brackets including the argument (the data we're passing in) in the order we wrote them in. So $num1 is set to 54 and $num2 is set to 83 within the function.

Variables used within a function, like $num1 and $num2 in the example, are only available within that function. As soon as that function ends, those variables are destroyed. They are called local variables and can only be accessed from within that function.

  • End Tutorial
  • Previous
  • Next

Variables in the main part of the code (not inside any functions) are global variables. We can use these within functions, but in order to do so, we need to import them into the function. For example, if we want to increment a number called $total which is a global variable within a function, we would need to do the following:

<?php
$total = 4;
function increment_total( ) {
   global $total;
   $total++;
}
?>

We use the keyword "global" to use $total in the function. Any changes we make to $total within the function will change the global variable for the whole program.
