Arrays

Arrays are a special type of variable. They are a group of similar variables organised into a structure.

  • End Tutorial
  • Next

In PHP, arrays are very flexible and allow many different uses. The basic usage of any array is as a list of items:

<?php
$temperatures = array( 13, 15, 12, 9, 11, 14, 8 );
?>

Here we have an array called $temperatures with 7 values (elements) in. This could be the past weeks' temperatures.

We can read each element out of the array. Arrays start counting from 0, so the first element is number 0, the last is number 6.

We can access the second element (which holds 15) by using the following code:

<?php
$second_day = $temperatures[1];
?>

As we've said, we want the second element. We know counting starts at 0, so the second element is number 1. We put 1 in square brackets after the variable name.

We can also use the same approach to write to array elements:

<?php
$temperatures[4] = 12;
?>

This sets the fifth element's value to 12.

You can also add a new element to the end, either by inserting that number:

<?php
$temperatures[7] = 9;
?>

Or by "pushing" it:

<?php
$temperatures[] = 9;
?>

By leaving the square brackets empty, we can "push" elements to the end of an array. This assigns this element to the next number (which is called an index).

PHP arrays resize automatically, so you can add as many elements as you like. You also don't need to add them in the right order. We could add a 58th element without having a 57th element.

With an array, what we are producing is an index/value list. We can say at the index "4" there is a value of "12", for instance.

Indexes in PHP can also be called "keys". So we can access any array value using its key.

Unlike other programming languages, keys don't have to be an integer, so we could map one string to another. For example, if translating, we could do this:

<?php
$spanish = array();
$spanish["hello"] = "hola";
$spanish["goodbye"] = "adios";
$spanish["world"] = "mundo";
?>

On a side note, we can create this array in one instruction in the following way:

<?php
$spanish = array(
   "hello" => "hola",
   "goodbye" => "adios",
   "world" => "mundo"
);
?>

We can access the Spanish translation of hello, goodbye and world now:

<?php
echo $spanish["hello"] . " " . $spanish["world"];
?>

This will output the Spanish equivalent of "Hello world" ("hola mundo").

As you can see, arrays are very useful in many different applications. Because keys are often integers, this is suited very well to a FOR loop. If we want to set all the values in an array, for instance:

<?php
$numbers = array();
$temp_num = 10;
for( $x=0; $x<10; $x++ ) {
   $numbers[$x] = $temp_num;
   $temp_num--;
}
?>

This will set 10 elements of the array with values from 10 down to 1.

  • End Tutorial

If we want to output all the values of an array, we can use a function called print_r:

<?php
print_r( $numbers );
?>

This will show you the whole contents of an array.

If we want to read array elements, we can use a FOR loop again. Another way we can do it is by using a FOREACH loop. These are designed just for arrays, and it means we can read all the elements out of the array:

<?php
foreach( $numbers as $value ) {
   $value . "\n";
}
?>
  • End Tutorial
  • Previous
  • Next

This takes each element in numbers. With each element it sets $value to this value then runs the code. It repeats this until it reaches the end of the array. It's very similar to using this FOR loop:

<?php
for( $x=0; $x< count ( $numbers ); $x++ ) {
   $value = $numbers[$x];
   //loop code in here
}
?>
  • End Tutorial

Although this loop code is less reliable than using the FOREACH (because non-numeric keys may be used, or the keys may not be in a sequence).

Using the FOREACH, we can get the key as well as the value, using the following code:

<?php
foreach( $numbers as $key => $value ) {
   $value . "\n";

We can also create "multi-dimensional arrays". Where a standard (one dimensional) array is like a list, two dimensional arrays are like a table, three dimensional arrays are like cubes. Arrays above three dimensions are difficult to visualise, but you can create an array with any number of dimensions.

  • End Tutorial
  • Previous
  • Next

To add an extra "dimension", we just place another array inside an array:

<?php
$table = array(
   array(53,68,85,13,74),
   array(86,46,57,53,24),
   array(95,24,11,67,88)
);
?>

This would create a two dimensional array with some numbers in it.

  • End Tutorial
  • Previous
  • Next

We could use two FOR loops (one within another) to create a two dimensional array and fill it with values:

<?php
$array = array();
$num=1;
for( $x=0; $x<5; $x++ ) {
   for( $y=0; $y<5; $y++ ) {
      $array[$x][$y] = $num;
      $num++;
   }
}
?>

If we wanted to access the number 57 in the middle of the table, we would do the following:

<?php
$num = $table[1][2];
?>

This is because it is in the second row (index 1) and the third column (index 2).
