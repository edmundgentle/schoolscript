Introduction

PHP is a programming language most commonly used when making websites. It is embedded within web pages and runs when the user asks for the web page.

It is mainly used for creating logins and user accounts, as well as commenting systems and website management tools. It is very simple to learn and use, which is why it's so popular. It is used by small businesses and very large businesses such as Facebook.

All instructions are run in the order in which they appear in the file. They start with the first instruction and then the code stops once the last instruction has finished running (executing).

We are going to write a script called "Hello World", which just writes out the words "Hello World" to the output panel (the box at the bottom of the window).

  • End Tutorial

  • Next

First, make sure that you have a PHP file open. Go to File Type in the top menu and select PHP from the list. In the code box to the left, write the following code:

<?php
echo "Hello World";
?>

Then press "Run". You should see "Hello World" in the output panel. As you write code, you can run it as you go by going to Run in the top menu then Run. It will (try and) run any code in the code box.
